## 知识点
###什么是时间复杂度
时间复杂度是对一个算法运行时间长短的量度， 用大O表示， 记作
T(n)=O(f(n))。

常见的时间复杂度按照从低到高的顺序， 包括O(1)、 O(logn)、 O(n)、
O(nlogn)、 O(n2)等。
###什么是空间复杂度
空间复杂度是对一个算法在运行过程中临时占用存储空间大小的量度， 用大O表
示， 记作S(n)=O(f(n))。

常见的空间复杂度按照从低到高的顺序， 包括O(1)、 O(n)、 O(n2)等。 其中递
归算法的空间复杂度和递归深度成正比。
